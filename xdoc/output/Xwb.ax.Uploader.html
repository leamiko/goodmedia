类 Xwb.ax.Uploader

空间：Xwb.ax
定义在:std-doc.js
类:Uploader
父类:Object
异步文件上传类，该类通过类似JSONP方式的隐藏IFRAME上传文件。IFRAME由类自动创建。 
X.use('Uploader', {
       form:$('#fileForm'),
       // 成功后回调 

       onload:function(e){
            if(e.isOk()){
                alert('uploaded!');
            }else {
                alert(e.getCode());
            }
       }
    });
