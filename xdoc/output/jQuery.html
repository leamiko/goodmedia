jQuery 高亮查找插件，完全基于文本结点的DOM操作，对原有非文本结点不造成任何影响。
Parameters:
  • keywords : Array
    高亮文本数组
  • [style] : String
    应用于高亮文本的样式类 用法: 
    高亮:
   $('#ss').highlight(['keyword', ... ]);
   清除:
   $('#ss').clearHighlight();
   
   或者传递自定义的样式类
   $('#ss').highlight(['keyword', ... ], 'cssClass');
   清除:
   $('#ss').clearhighlight('cssClass');
    请确保每次高亮前清除先前已高亮内容。 可选配置信息：$.fn.highlight.cls
